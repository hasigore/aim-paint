# Aim Contour 🎯

An interactive, browser-based mouse tracking and smoothness trainer designed to help FPS players calibrate their muscle memory, eliminate micro-stutters, and practice flawless angle contouring.

**Live Demo:** https://aimcontour.ddnsgeek.com/

---

## 🚀 Key Features

* **Geometric Precision Pathing:** Track your crosshair smoothly along precise geometric lines, figures, and shapes to build clean mouse control.
* **Custom Target Tracing:** Upload your own custom target images or SVG layouts directly in the browser to trace specific angles or custom gaming layouts.
* **100% Client-Side:** Runs completely in your local browser window using clean HTML5 Canvas and JavaScript. No installation, zero backend data tracking, and no lag.
* **Privacy First:** User-uploaded images are processed locally in your browser cache—they are never sent or stored on a web server.

---

## 🛠️ Built With

* **HTML5 Canvas** - Core rendering engine for geometric layouts.
* **Vanilla JavaScript** - High-polling mouse position capturing and path calculation.
* **CSS3** - Clean, minimalist, distraction-free user interface.
